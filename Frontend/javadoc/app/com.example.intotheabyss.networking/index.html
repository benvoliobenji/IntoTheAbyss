Package com.example.intotheabyss.networking

Types

Network

 class Network : Listener

This class is the implementation of the Kryonet library, which is our websocket implementation. The class, on invocation of connect(), creates listeners that grab packets from the server asynchronously and modify GameState to reflect the changes made by the server.

NetworkRunnable

 class NetworkRunnable : Runnable

A Network Thread that instantiates the Network and VolleyNetwork classes and creates/runs the UpdateThread. This is created because Android does not want any networking on the main thread

UpdateRunnable

 class UpdateRunnable : Runnable

An Update Thread that runs through a loop and calls UpdateVerification methods to make sure the server has the most up-to-date information from the client.
