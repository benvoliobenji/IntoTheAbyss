VolleyNetwork

class VolleyNetwork : VolleyNetworkInterface

This class is designed to implement VolleyNetworkInterface and to query the server using Volley. These conenctions are often used for files that are of larger size than what Kryonet can support and are also necessary for the client and server to recieve an ACK back based on the GET and PUT methods.
