UpdateVerification

 class UpdateVerification : UpdateVerificationInterface

UpdateVerification is a class that checks for differences between its values (x, y, floor) and those stored in GameState. If they are different, then UpdateVerification will call methods to the different networks to inform the server and get correct information to GameState.
