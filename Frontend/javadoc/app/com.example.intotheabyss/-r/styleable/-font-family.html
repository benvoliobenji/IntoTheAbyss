Attributes that can be used with a FontFamily.

Includes the following attributes:

AttributeDescription com.example.intotheabyss:fontProviderAuthorityThe authority of the Font Provider to be used for the request. com.example.intotheabyss:fontProviderCertsThe sets of hashes for the certificates the provider should be signed with. com.example.intotheabyss:fontProviderFetchStrategyThe strategy to be used when fetching font data from a font provider in XML layouts. com.example.intotheabyss:fontProviderFetchTimeoutThe length of the timeout during fetching. com.example.intotheabyss:fontProviderPackageThe package for the Font Provider to be used for the request. com.example.intotheabyss:fontProviderQueryThe query to be sent over to the provider.
