 class Player : PlayerInterface

The Player class implements the PlayerInterface and creates and object that will represent the User and the other heros the User will see when playing the app. This holds values necessary for display and game logic such as name, ID, coordinates, health, floor, and image to draw the player to the canvas.

The PlayerInterface. This is the interface that declares the functions the any class inheriting from PlayerInterface must implement. Designed to allow for dependency injection.
