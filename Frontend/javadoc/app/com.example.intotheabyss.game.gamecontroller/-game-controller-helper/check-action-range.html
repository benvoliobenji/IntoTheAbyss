fun checkActionRange(x: Float, y: Float, action: Int): Boolean

Overrides GameControllerHelperInterface.checkActionRange

Check if the action button was pressed. Checks if input coordinates fall in range of the action button. Also checks that the action status of the input wasn't ACTION_UP

Parameters

x - The x-coordinate of the input

y - The y-coordinate of the input.

action - The action status of the user input.

Return
A boolean value. True if an action has occurred. False if not.
