getAction

fun getAction(x: Float, y: Float, action: Int): Int

Overrides GameControllerInterface.getAction

Parse touch events to see if action button occurs. Will only accept a player action at a certain interval. Checks if player has touched in the action button range.

Parameters

x - x-coordinate of the user input

y - y-coordinate of the user input

action - the action status of the user input. If ACTION_UP, will not perform. Otherwise will.
