retrieveNewDungeonLevel

fun retrieveNewDungeonLevel(level: Int, network: Network): Unit

Overrides VolleyNetworkInterface.retrieveNewDungeonLevel

Sends a JSONObjectRequest to get the 2D array of the dungeon layout of the given level.

Upon a response, parse the grid with a utility method, place the stairs in the grid, and place the Player object in GameState at the spawn location of the level.

Parameters

level - The level number the User will be moving to.

network - The Kryonet instance to invoke kryonet methods (MoveFloorPacket).
