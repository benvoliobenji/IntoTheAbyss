drawPlayer

fun drawPlayer(dX: Int, dY: Int, context: Context, canvas: Canvas, player: Player, gAction: Int, isPlayer: Boolean): Unit

Overrides DrawPlayerInterface.drawPlayer

Function to draw a player to the canvas. Will call functions to update sprite for player. Does proper checks and then draws player to canvas. Draws player info if isPlayer is true.

Parameters

dX - the x-direction that the player is facing. May be updated from current state. -1 is left, 1 is right, 0 is idle

dY - the y-direction taht the player is facing. -1 is down, 0 is idle, 1 is up

context - The context so that we can access the game resources

canvas - The canvas object to draw the player to

gAction - The action status of the player we are drawing. 0 if no action, 1 if action

isPlayer - Is this player the owner of the phone? False if not, true if so.
